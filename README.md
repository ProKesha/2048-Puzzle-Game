# 2048 Puzzle Game

A browser-based implementation of the classic 2048 puzzle game built with vanilla JavaScript. The project focuses on game logic, DOM manipulation, and state management without frameworks.

## Live Preview

[Demo link](https://prokesha.github.io/2048-Puzzle-Game/)

## Overview

The goal of the game is to combine tiles with the same values until you reach the `2048` tile. Players use keyboard controls to move tiles across the board, while the game tracks the score and detects win or loss states.

## Technologies Used

- HTML5
- SCSS
- JavaScript (ES6)

## Features

- Full 2048 game logic implementation
- Keyboard controls with arrow keys
- Dynamic grid rendering
- Score tracking
- Win and game over detection
- Restart functionality
- Smooth tile animations

## Notes

This project was created to strengthen core JavaScript skills, including algorithms, event handling, and DOM manipulation.

## Getting Started

```bash
git clone https://github.com/ProKesha/2048-Puzzle-Game.git
cd 2048-Puzzle-Game
npm install
npm start
```
